Aspis Engine

The Lightweight Python Game Engine

Aspis is a modern, cross-platform 2D game engine built on Python and PySide6. Designed for rapid prototyping and flexibility, it empowers developers to create without limits.

Python Powered

Write your game logic in pure Python. Use the libraries you love like NumPy and Pygame seamlessly with full integration.

Visual Editor

Drag-and-drop scene composition. Inspect and modify properties in real-time with the built-in inspector panel.

ECS Architecture

Flexible Entity-Component-System design allows for modular and performant game code that scales with your project.

Physics Included

Integrated rigid body physics powered by Pymunk for realistic collisions, forces, and dynamic interactions.
